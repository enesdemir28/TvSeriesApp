# TvSeries App

Its a TvSeries application

## Tech Stack

- Android Studio
- Kotlin
- Retrofit
- MVVM
- Dagger/Hilt
- TvMaze API

## Description

You can view the list of TV series taken from TvMaze and filter them by name.
